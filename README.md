# BoardGame-ML-Prediction-Analysis
Developed a machine learning pipeline to analyse a dataset of 398 board games with over 460 features, including exploratory data analysis and predictive modelling for category, region, origin year, best AI agent, and UCT performance. The project includes the original dataset (CSV), a processed final dataset, and visualisations of key insights through graphs. Achieved high classification accuracy and identified structural features, such as branching factor and decision complexity, as key drivers of AI performance.
